# Getting started

	* copy Main.conf.example to Main.conf
	* edit Main.conf to fit login, admins, plugins, etc
	* run ./main.py
	* make sure you have tasbot module in python path

# Dependencies

	* http://pypi.python.org/pypi/trueskill
	* formalchemy, disqus-python
